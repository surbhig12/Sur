Testing with Samsung Connect

To test the ARTIK 530 or Raspberry Pi 3 boards you have developed with the Samsung Connect application:

  1. Launch the Samsung Connect application.
    Note The Samsung Connect application is preloaded in Galaxy S8 and Galaxy Note 8 phones. In addition, if the phone is based on Android 6.0 or later, you can download Samsung Connect from the Google Play store. The Samsung Connect version must be 1.6.6-05 or higher.
  2. In the Dashboard, select Settings (3 dots at the top right) > Settings and long-press About Samsung Connect more than 5 seconds.

    DEVELOPER MODE appears below About Samsung Connect.

  3. Turn Developer mode on.
  4. In the Devices tab, select ADD.
  5. Find the deviceName defined in the JSON file and select it.

After the above setup process, you can control the device.

If you want to create your controller in the Samsung Connect application, see https://smartthings.developer.samsung.com/develop/guides/ocf/quickstart.html.
